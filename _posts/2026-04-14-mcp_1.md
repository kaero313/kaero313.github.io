# MCP
---

MCP(Model Context Protocol)는 한 마디로 **AI 모델과 외부 데이터/도구를 연결하는 표준 규격**이다.
앤트로픽에서 정의한, AI가 사용하는 외부 도구 공용 인터페이스 표준이라고 볼 수 있다.

## 통신 방식

통신 방식에는 2가지가 있다. 
- 로컬(Standard I/O)
- 원격(HTTP/SSE)

| 구분 | 로컬(stdio) | 원격 (HTTP/SSE) |
| :--- | :--- | :--- |
| **개념** | 로컬 프로세스 간 표준 입출력 통신 | 웹 프로토콜 기반의 원격 통신 |
| **연결 방식** | `stdin` / `stdout` 파이프라인 | `HTTP POST` + `Server-Sent Events` |
| **서버 위치** | 사용자 로컬 컴퓨터 (Local) | 원격 서버 또는 클라우드 (Remote) |
| **실행 주체** | AI 호스트 앱이 서버를 직접 실행 | 별도의 독립된 서버 프로세스가 상시 구동 |
| **주요 장점** | 보안성 우수, 설정 간편, 저지연(Low Latency) | 다수 유저 공유 가능, 클라우드 연동 용이 |
| **보안** | 로컬 권한 내에서 격리됨 | 네트워크 보안(CORS, 인증) 설정 필요 |
| **적합한 사례** | 개인 파일 분석, 로컬 스크립트 실행 | 팀 공용 DB 조회, 외부 API 서비스 통합 |
