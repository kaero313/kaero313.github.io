# MCP
---

MCP(Model Context Protocol)는 한 마디로 **AI 모델과 외부 데이터/도구를 연결하는 표준 규격**이다.
앤트로픽에서 정의한, AI가 사용하는 외부 도구 공용 인터페이스 표준이라고 볼 수 있다.

## 통신 방식

통신 방식에는 2가지가 있다. 
- 로컬(Standard I/O)
- 원격(HTTP/SSE)

| 구분 | 로컬(stdio) | 원격 (HTTP/SSE) |
| :--- | :--- | :--- |
| **개념** | 로컬 프로세스 간 표준 입출력 통신 | 웹 프로토콜 기반의 원격 통신 |
| **연결 방식** | `stdin` / `stdout` 파이프라인 | `HTTP POST` + `Server-Sent Events` |
| **서버 위치** | 사용자 로컬 컴퓨터 (Local) | 원격 서버 또는 클라우드 (Remote) |
| **실행 주체** | AI 호스트 앱이 서버를 직접 실행 | 별도의 독립된 서버 프로세스가 상시 구동 |
| **주요 장점** | 보안성 우수, 설정 간편, 저지연(Low Latency) | 다수 유저 공유 가능, 클라우드 연동 용이 |
| **보안** | 로컬 권한 내에서 격리됨 | 네트워크 보안(CORS, 인증) 설정 필요 |
| **적합한 사례** | 개인 파일 분석, 로컬 스크립트 실행 | 팀 공용 DB 조회, 외부 API 서비스 통합 |

상세한 설명은 GPT에게 맡기고, 난 [korean-law-mcp](https://github.com/chrisryugj/korean-law-mcp) 를 활용한 온프레미스 및 외부 연동 실무 구현에 집중해보겠다.

## 로컬(Stdio) 방식

내 컴퓨터를 서버로 하여 사용하는 방식이다. 로컬로 하게 된다면 보통 CLI나, IDE에서 사용하게 될 것이다.

### CLI에서 사용

우선 설치가 필요하다. 해당 mcp는 node 기반이기 때문에 npm으로 설치를 진행한다. 그 후, 법제처 Open API 인증키를 설정해준다.

```bash
# 설치
npm install -g korean-law-mcp

# 인증키 설정 (honggildong을 본인 키로 바꾸세요)
export LAW_OC=honggildong        # Mac/Linux
set LAW_OC=honggildong           # Windows CMD
$env:LAW_OC="honggildong"       # Windows PowerShell

# 사용 예시
korean-law "민법 제1조"                    # 자연어로 바로 조회
korean-law search_law --query "관세법"     # 도구 직접 호출
korean-law list                            # 전체 도구 목록
korean-law list --category 판례            # 카테고리별 필터
korean-law help search_law                 # 도구별 도움말
```
